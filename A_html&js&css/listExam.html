정렬된 목록(숫자형)
  1. 티포트에 드라이 허브 3g을 넣는다.
  2. 끓는 물 200ml를 붓고 뚜껑을 닫아 3~5분 정도 우려낸다.
    허브티는 꼭 끓는 물로 우리는 것이 포인트다.
  3. 거름망을 이용해 찻잔에 따른다.
정렬된 목록(대문자)
  1. 티포트에 프레시 허브를 잘게 찢어 넣는다.
  2. 드라이 허브(3g)를 사용할 때의 2~3배 양이 적당하다.
  3. 끓는 물 200ml를 붓고 뚜껑을 닫아 3~5분 정도 우려낸다.
  4. 거름망을 이용해 찻잔에 따른다.
